Русские шашки (Russian draughts)

Ходы выполняются вводом в консоль последовательности полей, 
которую должна пройти шашка/дамка

Например: a1c3a5d8

---------------------------------------------------

Для того, чтобы начать играть, исполните этот скрипт
в корне проекта:

./run.sh
